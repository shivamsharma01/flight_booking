Install mysql DB and create a schema

To run the Backend
	install python
	install mysql connector, flask, flask_cors
	provide username, password and schema name of the database to the connection object
	run the file
	
To run the UI
	install npm, node, angular cli
	open cmd abd cd ui-angular folder
	run npm install
	run ng serve command
	open browser and go to 'localhost:4200'
	ui is configure with the address 'http://127.0.0.1:5000/flightbooking/*' please update the port if changed in python file
